# orbital_motion_simulator

# Orbital Motion Simulator Version 1

A 2D orbital mechanics simulator written in Python using NumPy and Matplotlib.

## Features
- Circular orbit simulation
- Elliptical orbit simulation
- Escape trajectory simulation
- Earth impact trajectory simulation
- Euler-Cromer numerical integration
- Static visualization
- Animated GIF visualization

## Physics
The simulator models gravity using:

F = GMm/r²

and updates motion using the Euler-Cromer integration method.

## Technologies
- Python
- NumPy
- Matplotlib
